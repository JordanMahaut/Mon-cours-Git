Mon cours sur GIT

Instruction pour GIT

  • Installer Git sur internet
  • Ouvrir l'executeur Git Bash
  • Accéder avec le "cd ~" dans le bon répertoire
  • Une fois le code HTML, CSS, JavaScript... Exécuter les commandes ci dessous (Si besoin installer Git-It pour voir l'ordre des commandes)

Instruction sur GitHub

  • Cliquez sur "Your repositories" à partir de notre icon en haut à droite
  • Cliquez sur "New" pour créer notre répertoire de dépot
  • Entrée le nom du répertoire et choisir si il est en "Public" ou "Private" puis cliquez sur Create repository

Git commit Git commit –m “Description du commit” La commande git commit permet de valider les modifications apportées au HEAD. Notez que tout commit ne se fera pas dans le dépôt distant.
Git status Git status La commande git status affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés.
Git push Git push origin master Git push est une autre commandes GIT de base. Un simple push envoie les modifications locales apportées à la branche principale associée
Git checkout Git checkout La commande git checkout peut être utilisée pour basculer des branches entre elles.
